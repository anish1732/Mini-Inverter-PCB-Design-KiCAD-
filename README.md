# Mini Inverter PCB Design

This repository contains the complete KiCad project files, schematics, PCB layout, and fabrication outputs for a **mini inverter circuit**.  
The design converts DC input into AC output using MOSFET switching and driver ICs, optimized for compact size and efficient performance.

---

## 📂 Repository Contents
- **/design** → KiCad schematic (`.sch`) and PCB layout (`.kicad_pcb`)
- **/fabrication** → Gerber files for PCB manufacturing
- **/bom** → Bill of Materials (BOM) for component sourcing

---

## ⚡ Features
- Two-layer PCB with copper fills for improved grounding and thermal management
- MOSFET-based inverter topology with driver ICs
- DRC/ERC checks performed for manufacturability
- Ready-to-fabricate Gerber outputs
- Open-source hardware license for community use and modification

---

## 🛠️ How to Use
1. Clone or download this repository.
2. Open the KiCad project in **KiCad v7 or later**.
3. Run **ERC/DRC checks** to verify design integrity.
4. Use the provided Gerber files for PCB fabrication.
5. Refer to the BOM for component sourcing.
